This folder contains:

DATA		A subfolder containing the property price indices as well as the input files used by the algorithm
SHAPES		A subfolder containing shapefiles that can be used to map the property price indices
XWALKS		A subfolder containing crosswalks from postcodes to counties and local labour markets, as well as postcode population that may be used for weighted aggregation

The spatial units are defined as follows in both folders

LLM		Local labor markets as defined by Kosfeld and Werner (2012)
COUNTIES	Kreise & Kreisfreie Städte (NUTS3 regions)
VWG		Municipal associations (Verbandsgemeinden)
POSTCODES	Five-digit postal codes
